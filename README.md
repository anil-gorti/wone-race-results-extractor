# WONE — Race Results Extractor

Extract participant data from race timing platforms with adaptive parsing.

Paste one or more race result URLs and get clean, structured outputs like race name, category or distance, and finish time. Built for runners, coaches, and analysts who want a reliable way to consolidate race history across inconsistent timing websites.

Features
	•	URL-based extraction from race timing pages
	•	Adaptive parsing across different site structures
	•	Batch processing for multiple links
	•	Real-time processing status
	•	Clean, CSV-ready structured output

Use Case

Instead of manually searching each race website, paste all your result links and generate a unified dataset of your race history in seconds.
